Architecture MVC

  • Couplage faible et base extensible.
// Contrôleur/Service (Singleton)
public final class SessionManager {
  private static final SessionManager INSTANCE = new SessionManager();
  private final Map sessions = new HashMap<>();
  private SessionManager() {}
  public static SessionManager getInstance() { return INSTANCE; }
  public Session create(Course c, LocalDate s, LocalDate e, int max, Teacher t) {
    Session ses = new Session(c, s, e, max, t);
    sessions.put(ses.getId(), ses);
    c.addSession(ses);
    return ses;
  }
}

Design Patterns Utilisés
  • State: contrôle des états de Session.
  • Observer: notifications aux étudiants et formateur.
  • Singleton: CourseManager, SessionManager.
  • Justification: clarté, isolation et simplicité.
// State Pattern
public interface SessionState {
  String getNotifyMessage();
  default void subscribe(Session s, Student st) { throw new IllegalStateException(); }
  default void unsubscribe(Session s, Student st) { throw new IllegalStateException(); }
  default void start(Session s, Teacher t) { throw new IllegalStateException(); }
  default void cancel(Session s, Teacher t) { throw new IllegalStateException(); }
}

// Exemple d'état
public class OpenSessionState implements SessionState {
  public String getNotifyMessage() { return "Session ouverte"; }
  public void start(Session s, Teacher t) { s.setState(new StartedSessionState(t)); }
}

Flux d’Inscription & Waitlist
  • Si capacité: inscription immédiate.
  • Si pleine: ajout à la waitlist (FIFO).
  • Libération d’une place: promotion automatique.
  • Notifications lors de la promotion.
// Session.subscribe (extrait)
public void subscribe(Student student) {
  if (isSubscribed(student.getId())) throw new IllegalStateException("Déjà inscrit");
  if (isFull()) { addToWaitlist(student); return; }
  students.put(student.getId(), student);
  addObserver(student);
  inscriptions.put(student.getId(), new Inscription(student, this));
  if (isFull()) { setState(new FullSessionState()); notifyObservers("Session pleine"); }
}

private void promoteFromWaitlist() {
  if (!isFull() && !waitlist.isEmpty()) {
    Student next = waitlist.poll();
    subscribe(next);
    notifyObservers("Promotion d'attente: " + next.getName());
  }
}

Notifications (Observer)

  • Événements: changement d’état, session pleine, promotion.

// Enregistrement des observers
public Session(Course c, LocalDate s, LocalDate e, int max, Teacher t) {
  this.course = c; this.startAt = s; this.endAt = e; this.maxPlaces = max; this.teacher = t;
  addObserver(t); // formateur notifié
}

public void setState(SessionState state) {
  this.state = state;
  notifyObservers(state.getNotifyMessage());
}
